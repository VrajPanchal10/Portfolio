## 📖 About The Project

This is a **single-page portfolio** showcasing projects and skills. Built with vanilla HTML, CSS, and JavaScript (no frameworks), featuring clean glassmorphism design, smooth animations, and full responsiveness across all devices.

---

## ✨ Features

- 🎨 **Dark/Light Mode** - Toggle theme with auto-save
- ✍️ **Typing Animation** - Dynamic role showcase
- 📱 **Fully Responsive** - Mobile, tablet, and desktop
- 🎭 **Smooth Animations** - Scroll reveals and hover effects
- 📋 **Tabbed About Section** - Skills, Experience, Education
- 🔍 **Project Filter** - Category-based project filtering
- 📧 **Working Contact Form** - EmailJS integration
- 🎯 **Smooth Scrolling** - Anchor link navigation
- 📱 **Mobile Menu** - Hamburger navigation for small screens
